Version de Qt utlisée : Qt Creator 4.2.1 sous Windows 10 (Basé sur Qt 5.8.0 avec MSVC 2015, 32 bit)

Pour pouvoir jouer :
1 - Ouvrir le fichier "SuperMarMan.pro" à partir du dossier SuperMarMan afin d'ouvrir le projet
2 - Ouvir dans l'éditeur l'ensemble des fichiers .h et .cpp
3 - Exécuter pour compiler (ou CTRL + R)
4 - Enjoy :)
